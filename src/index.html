Login

On the official Simcast website, the login page is located on the same page as all the news articles which can be very confusing and distracting to ADHD users. To make this website ADHD friendly, the login page has been place on a separate dedicated space to ensure that users have no distractions whilst they are logging in. On a separate webpage there is enough space to provide users with clear, text-based error messages and prompts ensuring that ADHD users know exactly what steps they should be taking.

Users with ADHD work better with text-based error messages that are eye-catching and can be understood easily so that the user can correct the issue promptly. It is important to specify what the exact issue is. For example, if a username is invalid there should a prompt suggesting why this could be so, perhaps indicating that the username is strictly an email.

Your username will be your email address

It would also be useful to add a graphical password authentication (GPA) system rather than the conventional password method. For this, users have to select a set of images in a specific order in order to access their account. There are other alternatives to GPA and these include a question based password or pin codes. Stimulating the brain is essential to help ADHD users memorise information therefore, this 'unconventional' authentication method could prove to be rather useful.

Select the images in the correct order

The official website contains a conventional password method that can easily be forgotten by those with ADHD. Instead one could implement a graphical password authentication (GPA) system so that ADHD users are more likely to remember the unconventional password.

Sign In

On the official Simcast website the sign in and sign up button can hardly be distinguished from one another. To solve this problem the two buttons have been presented in two different colours that can be easily be distinguished by an ADHD user.

It is essential that users can hide the tool bar in order to prevent further distraction.
